# Evaluation and Acceptance Mode / 评估与验收模式

## Trigger

- User asks how to test, evaluate, accept, or launch the AI or Agent feature

## Required sections

- User value metrics / 用户价值指标
- AI quality metrics / AI 质量指标
- Experience metrics / 体验指标
- Operations metrics / 运营指标
- Agent execution metrics if applicable / Agent 执行指标
- Test sample design / 测试样本设计
- Pass criteria / 通过标准
- Human evaluation mechanism / 人工评估机制
- Retest frequency / 复测频率
- Definition of Ready / Definition of Done / DoR 与 DoD
