# PRD Review Mode / PRD 评审模式

## Trigger

- User provides an existing PRD, solution draft, or requirement document for review

## Review output

- Overall readiness / 整体成熟度
- Key gaps / 关键缺口
- AI boundary gaps / AI 能力边界缺口
- Agent design gaps if applicable / Agent 设计缺口
- Functional rules gaps / 功能规则缺口
- Data loop, tracking, analytics, admin gaps / 数据闭环、埋点、分析、管理端缺口
- Risk and fallback gaps / 风险与兜底缺口
- Anti-pattern warnings / 反模式提醒
- Priority-ranked fixes / 优先级修复建议
- Suggested replacement text / 建议替换文案
