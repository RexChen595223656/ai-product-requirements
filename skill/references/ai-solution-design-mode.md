# AI Solution Design Mode / AI 方案设计模式

## Trigger

- Prompt, RAG, model selection, knowledge base, or AI workflow questions

## Required sections

- AI task definition / AI 任务定义
- Input range / 输入范围
- Output format / 输出格式
- Processing logic / 处理逻辑
- Prompt strategy / Prompt 策略
- Knowledge base strategy / 知识库策略
- Data strategy / 数据策略
- Explainability and editability / 可解释与可编辑
- Failure strategy / 失败策略
- Evaluation metrics / 评估指标

## Checks

- Do not write only "call the LLM".
- Define model responsibility level.
