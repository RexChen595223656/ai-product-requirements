# Functional Rules Mode / 功能规则模式

## Trigger

- Convert ideas, PRDs, or AI designs into buildable rules

## Required sections

- Functional goal / 功能目标
- Actors / 参与角色
- Preconditions / 前置条件
- Main flow / 主流程
- Business rules table / 业务规则表
- AI or Agent behavior rules / AI 或 Agent 行为规则
- Exception rules / 异常规则
- State transitions / 状态流转
- Permission and data rules / 权限与数据规则
- Given / When / Then acceptance / 验收条件
