# Role-specific Outputs / 角色化输出

## Output options

- Executive brief / 管理层摘要
- Product brief / 产品摘要
- Design brief / 设计摘要
- Engineering spec / 工程说明
- AI and data spec / AI 与数据说明
- QA checklist / QA 清单
- Operations playbook / 运营手册

## Rule

Keep the same facts across versions. Change emphasis, detail level, and terminology for the audience.
