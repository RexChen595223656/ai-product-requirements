# Tracking Requirements Mode / 埋点需求模式

## Trigger

- User asks for events, instrumentation, or tracking specs

## Required sections

- Tracking goals / 埋点目标
- Metrics-to-events mapping / 指标到事件映射
- Event table / 事件表
- Properties / 属性
- Reporting owner / 上报方
- Deduplication and exception rules / 去重与异常规则
- Dashboard mapping / 看板映射
- Given / When / Then acceptance / 验收

## Rules

- Event names and property keys stay in English.
- User-facing labels may be bilingual.
