# Requirement Clarification Mode / 需求澄清模式

## Trigger

- The user explicitly asks for a structured clarification memo
- A requirement-style deliverable is requested, but one narrow missing piece still blocks a useful document
- The goal is to output a short decision memo, not to run full open-ended ideation

## Output

### 中文

1. 当前成熟度判断
2. 这个问题是否适合用 AI
3. 可能的用户角色与场景假设
4. 当前最关键的缺失信息
5. 1-3 个澄清问题
6. 下一步最小动作

### English

1. Maturity judgment
2. Whether AI is appropriate
3. Likely user and scenario hypotheses
4. Key missing information
5. 1-3 clarifying questions
6. Next smallest action

## Boundary

- Do not use this mode as a substitute for generic brainstorming.
- If the user is still exploring the space broadly, a process skill should lead first.
