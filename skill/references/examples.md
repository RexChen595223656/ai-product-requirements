# Examples / 示例

## Good prompt example

`帮我 review 这份 AI PRD，重点看功能规则、埋点、数据闭环和风险兜底是否完整。`

Expected mode: PRD review
Expected format: Chinese-only headings

## Good prompt example

`Is this better as a normal AI feature or an Agent? The workflow needs tool calls across CRM and ticketing systems.`

Expected mode: Agent design
Expected format: English-only headings

## Good prompt example

`帮我把这个 TK 视频功能想法拆成需求分析`

Expected mode: Requirement analysis
Expected format: Chinese-only headings such as `需求分析`、`目标用户`、`核心场景`

## Anti-example

`用户角色 / User Role`

Why it is weak: this is a bilingual teaching-style heading, not a natural default heading for Chinese output.
