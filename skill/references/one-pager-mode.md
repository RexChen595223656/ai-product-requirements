# One-pager Mode / 一页纸模式

## Trigger

- Leadership alignment
- Review meeting summary
- Cross-functional kickoff summary

## Output structure

- 一句话定位 / One-sentence positioning
- 目标用户 / Target users
- 核心场景 / Core scenario
- 核心痛点 / Core pain
- 核心价值 / Core value
- 成功指标 / Success metrics
- 版本边界 / Version boundary
- 风险 / Risks
- 决策点 / Decision points
