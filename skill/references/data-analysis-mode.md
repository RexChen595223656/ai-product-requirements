# Data Analysis Mode / 数据分析模式

## Trigger

- User asks how to evaluate product effects after launch
- User asks for dashboard, metric hierarchy, or diagnosis

## Required sections

- Analysis goal / 分析目标
- North-star metric / 北极星指标
- Metric hierarchy / 指标分层
- Dimensions / 分析维度
- Dashboard structure / 看板结构
- Diagnosis framework / 诊断框架
- Optimization actions / 优化动作
