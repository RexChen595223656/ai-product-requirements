# Data Recycling Mode / 数据回收模式

## Trigger

- User asks what data to collect after launch
- User asks how to improve AI over time

## Required sections

- Recycling goal / 回收目标
- Data sources / 数据来源
- Data items / 数据项
- Trigger points / 采集时机
- Usage / 用途
- Sensitivity / 敏感等级
- Processing rules / 处理规则
- Retention and access / 保存与访问
- Labeling and quality control / 标注与质控
- Optimization path / 回流路径
- Operations loop / 运营闭环
