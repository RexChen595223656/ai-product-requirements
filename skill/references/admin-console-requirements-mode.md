# Admin Console Requirements Mode / 管理端需求模式

## Trigger

- Operations, review, moderation, knowledge management, config, audit, intervention, or handoff needs

## Required sections

- Console goal / 管理端目标
- Roles and permissions / 角色与权限
- Modules / 模块
- List page requirements / 列表页要求
- Detail page requirements / 详情页要求
- Configuration capabilities / 配置能力
- Review and intervention workflow / 审核与干预流程
- Dashboards / 看板
- Logs and audit / 日志与审计
- Alerts and exception handling / 告警与异常处理
- Acceptance criteria / 验收条件
