# Launch Readiness Mode / 上线准备模式

## Trigger

- User prepares beta, GA, rollout, operations, or launch review

## Required sections

- Rollout plan / 发布计划
- Tracking checklist / 埋点清单
- Dashboard checklist / 看板清单
- Beta pass criteria / Beta 通过标准
- GA pass criteria / GA 通过标准
- Operations workflow / 运营流程
- Feedback loop / 反馈闭环
- Knowledge, prompt, model optimization loop / 知识库、Prompt、模型优化闭环
- Launch risk checklist / 上线风险清单
