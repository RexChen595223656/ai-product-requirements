# Risk Fallback Mode / 风险兜底模式

## Trigger

- User asks about hallucination, safety, permissions, low confidence, sensitive data, or fallback

## Required sections

- Risk list / 风险清单
- Trigger conditions / 触发条件
- User messaging / 用户提示
- System behavior / 系统行为
- Human intervention rules / 人工介入规则
- Monitoring metrics / 监控指标
- Owner / 责任人
- Disable or degrade strategy / 降级或关闭策略
